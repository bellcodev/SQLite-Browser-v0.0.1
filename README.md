
# SQLite Browser v0.0.1

![SQLite Browser](https://i.imgur.com/1NAP2ZD.png)

## Descripción (Español)

*SQLite Browser v0.0.1* es una aplicación de escritorio desarrollada en *Python* que permite a los usuarios interactuar con bases de datos *SQLite* de manera visual e intuitiva, sin necesidad de escribir código SQL manualmente. 

Esta herramienta está diseñada para estudiantes, desarrolladores principiantes y cualquier persona que quiera explorar, crear y modificar bases de datos SQLite de manera rápida y sencilla.

*Características principales:*
- Crear nuevas bases de datos SQLite desde la interfaz.
- Crear, eliminar y modificar tablas fácilmente.
- Insertar, actualizar y eliminar registros de manera visual.
- Ejecutar consultas SQL personalizadas.
- Interfaz gráfica amigable con botones y menús intuitivos.

*Tecnologías utilizadas:*
- Python 3.x
- Tkinter (interfaz gráfica)
- SQLite (base de datos)
- Markdown para documentación y README
- Git / GitHub para control de versiones

*Estado del proyecto:*
- Versión alfa (v0.0.1)
- Funcionalidades básicas implementadas
- Próximas mejoras: soporte para exportar/importar bases de datos, filtros avanzados y mejora de UI.

---

## Description (English)

*SQLite Browser v0.0.1* is a desktop application developed in *Python* that allows users to interact with *SQLite* databases visually and intuitively, without manually writing SQL code.

This tool is designed for students, beginner developers, and anyone who wants to explore, create, and modify SQLite databases quickly and easily.

*Main Features:*
- Create new SQLite databases from the interface.
- Easily create, delete, and modify tables.
- Insert, update, and delete records visually.
- Execute custom SQL queries.
- User-friendly graphical interface with intuitive buttons and menus.

*Technologies Used:*
- Python 3.x
- Tkinter (GUI)
- SQLite (database)
- Markdown for documentation
- Git / GitHub for version control

*Project Status:*
- Alpha version (v0.0.1)
- Basic functionalities implemented
- Future improvements: database export/import, advanced filters, improved UI.
